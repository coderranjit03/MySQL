## 💡 Example SQL Snippets

### **Select Query**

```sql
SELECT first_name, last_name
FROM employees
WHERE department = 'Sales';
```

### **Aggregate Function**

```sql
SELECT department, COUNT(*) AS emp_count
FROM employees
GROUP BY department;
```

### **Join Example**

```sql
SELECT e.first_name, d.department_name
FROM employees e
JOIN departments d ON e.department_id = d.department_id;
```
